# EdgeScribe – AI Clinical Consultation Platform

A full-stack web platform built for student doctors to record, transcribe, and structure live Hinglish (Hindi-English) consultations.

## Features
- **Live Voice Recording** – Captures consultations via browser microphone using Web Speech API
- **Hinglish Transcription** – Real-time transcription of code-switched Hindi-English medical speech
- **Auto SOAP Note Generation** – AI pipeline maps raw transcript to structured Subjective, Objective, Assessment, Plan fields with source-sentence traceability
- **MedLedger Audit Trail** – SHA hash-based immutable record system; doctor-signed notes are cryptographically locked and cannot be edited post-approval
- **Doctor Sign & Approve Workflow** – Confirmation-based signing system with immutable badge marking

## Tech Stack
JavaScript, HTML, CSS, Web Speech API, GSAP, SHA Hashing, localStorage
